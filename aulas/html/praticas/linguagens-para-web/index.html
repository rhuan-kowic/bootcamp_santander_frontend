Linguagens para web

As tres linguagens aprendida no bootcamp Santander Frontend são:

  1. HTML
  2. CSS
  3. JavaScript

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação fundamental para a criação e estrutura de páginas da web.

CSS

CSS (Cascading Style Sheets) é a linguagem que define a apresentação visual de um documento HTML, como cores, fontes, espaçamentos e layouts

JavaScript

JavaScript (JS) é uma linguagem de programação de script versátil e amplamente utilizada, especialmente para criar páginas web interativas.

Os principais frameworks JavaScript, que são ferramentas que facilitam o desenvolvimento de aplicações web, são:

  • React
  • Angular
  • Vue.js

React

O React é um framework JavaScript de código aberto, criado e mantido pelo Facebook, focado em criar interfaces de usuário. É conhecido pela sua flexibilidade e capacidade de construir componentes reutilizáveis.

Angular

O Angular é um framework JavaScript de código aberto, também, um framework de código aberto, geralmente utilizado em aplicações de larga escala.

Vue.js

O Vue.js é um framework JavaScript de código aberto, conhecido por ser simples e fácil de aprender, sendo ideal para projetos que exigem velocidade e simplicidade.
