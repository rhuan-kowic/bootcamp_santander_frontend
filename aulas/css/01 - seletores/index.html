Bem-vindo!

No CSS nós possuimos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML específica.
  • Seletor por ID (#): Busca elementos através do atributo "id".
  • Seletor por Classe (.): Busca elementos através do atributo "class".
  • Seletor Universal (*): Seleciona todos os elementos HTML.
  • Seletor de Atributo([atrib]): Seleciona todos os elementos HTML que possuem um atributo específico em nosso documento HTML e conseguimos tambem buscar atricutos com um valor especifico.
